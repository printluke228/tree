第二章 相交线与平行线

1. 相交线
对顶角：相等
邻补角：和为 180
垂直：过一点有且只有一条直线与已知直线垂直
垂线段：点到直线的距离是垂线段的长度
2. 平行线
判定：同位角相等 / 内错角相等 / 同旁内角互补 → 两直线平行
性质：两直线平行 → 同位角相等 / 内错角相等 / 同旁内角互补
3. 平移
性质：对应点连线平行且相等，图形形状、大小不变

第三章 变量之间的关系

1. 变量与常量
变量：变化的量；常量：不变的量
2. 表示方法
表格法：清晰展示数值对应关系
关系式法：用数学式子表示变量关系
图象法：直观反映变化趋势
3. 常见变化类型
匀速变化：图象为直线
分段变化：图象为折线

第四章 三角形

1. 三角形基本性质
三边关系：两边之和大于第三边，两边之差小于第三边
内角和：\(180^\circ\)
外角：等于不相邻两内角之和
2. 三角形分类
按角分：锐角、直角、钝角三角形
按边分：不等边、等腰、等边三角形
3. 全等三角形
判定：SSS / SAS / ASA / AAS / HL（直角三角形）
性质：对应边相等，对应角相等
4. 尺规作图
作一条线段等于已知线段
作一个角等于已知角

第五章 生活中的轴对称

1. 轴对称图形
定义：沿一条直线折叠后，直线两旁的部分能完全重合
常见图形：等腰三角形、矩形、菱形、圆
2. 轴对称性质
对应点连线被对称轴垂直平分
对应线段、对应角相等
3. 等腰三角形
性质：两腰相等，两底角相等，三线合一
判定：两边相等 / 两角相等的三角形
4. 镜面对称
镜子中的图象与原图象关于镜面对称

第六章 概率初步

1. 事件分类
必然事件：一定发生，概率为 1
不可能事件：一定不发生，概率为 0
随机事件：可能发生也可能不发生，概率在 0~1 之间
2. 概率计算
古典概型：\(P(A) = \frac{事件A包含的结果数}{所有可能的结果数}\)
频率估计概率：大量重复试验时，频率稳定在概率附近
3. 游戏公平性
判断标准：双方获胜的概率是否相等
