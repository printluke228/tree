# 项目主入口：index.py（Python/Flask）
本项目核心运行文件为`index.py`，`index.html`为页面文件（同时作为GitHub Pages临时静态入口）。

## 本地运行（核心方式）
1. 安装依赖：pip install flask
2. 启动服务：python index.py
3. 浏览器访问：http://127.0.0.1:5001（加载index.html内容）

## 文件说明
- 🚀 index.py ：Flask主程序（项目核心，必须运行这个文件）
- 📄 index.html ：页面内容文件（Flask自动加载，同时为GitHub Pages静态入口）
